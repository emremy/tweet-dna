# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can use any image editing tool to create these, or use an online icon generator.

For a quick placeholder, you can use a simple X logo or any square image resized to these dimensions.
